# Nebula / laozig 主题二开说明

NodeGet-StatusShow 的深度二开。两轮迭代,全部通过 `npm run typecheck` + `npx vite build` 验证。
本地预览:`npm run dev`(数据驱动界面需在 `.env.local` 配好后端;主题/右键菜单等不依赖数据)。

---

## 第五轮(本次)

- **真·多风格主题(8 套)**:不再只是换色。每套换 **字体 / 圆角 / 调色板 / 质感**。
  - Tailwind `font-sans`/`font-mono` 改走 CSS 变量,每个 `[data-theme]` 覆盖。
  - 加载 5 个 Google 字体(Cinzel / Orbitron / Share Tech Mono / Special Elite / VT323)。
  - 套系:**简洁现代**(留白大圆角)、**科技**(等宽+网格底)、**未来**(Orbitron+霓虹辉光)、**像素**(VT323+0圆角+硬投影+CRT扫描线)、**写实**(实色分层柔影)、**古朴**(中文衬线+羊皮纸暖褐)、**中世纪**(Cinzel刻铭+暗石烫金)、**叙利亚战损**(打字机+做旧虚线裂边+噪点)。已浏览器实测像素/中世纪迥异。
- **卡片加单机累计总流量**:每张卡片新增「总流量 ↓累计接收 ↑累计发送」(本机,非全局)。
- **CPU 趋势加宽加高**:`h-5` 细线 → `h-10` 渐变面积图 + 区间/当前值标注。
- **移动端表格转卡片堆叠**:`<md` 时表格视图改为竖向卡片列表(状态胶囊+三指标+网速+月费)。
- **完整 PWA 离线**:[public/sw.js](public/sw.js) service worker(导航 network-first 离线回退、同源资源 SWR),main.tsx 生产环境注册。



**修复(均已连真实后端浏览器实测):**
- **Ping 图不再糊**:[NodeDetail.tsx](src/components/NodeDetail.tsx) 多来源时只画**清爽彩色线**(去面积填充),单来源才填渐变;加淡网格线、图表加高。
- **标题改 laozig**:`.env.local`、index.html、nodeget-theme.json 全部改为 laozig;`document.title` 动态跟随站名。
- **地图脉冲生效**:[WorldMap.tsx](src/components/WorldMap.tsx) `setOption(notMerge:true)` 避免旧 map-series 合并残留;深色底图 + 强调色脉冲光点已正常显示。

**全上(新功能,均已实测):**
- **键盘快捷键** [KeyboardShortcuts.tsx](src/components/KeyboardShortcuts.tsx):`/`或`⌘K`搜索、`1/2/3`切卡片/表格/地图、`t`主题、`b`费用、`Shift+D`明暗。
- **节点收藏置顶** [usePins.ts](src/hooks/usePins.ts):右键「置顶」、卡片图钉标记 + 高亮、置顶节点排最前(localStorage 记忆)。
- **卡片 CPU 趋势火花线** [NodeCard.tsx](src/components/NodeCard.tsx):每张卡片显示近期 CPU 走势。
- **浏览器原生通知** [NodeStatusWatcher.tsx](src/components/NodeStatusWatcher.tsx):节点离线时发系统通知(按需申请权限)。
- **代码分割**:懒加载 NodeDetail/BillingPage,**主包 663KB→231KB**(gzip 192→74KB),recharts/echarts 按需加载。
- **PWA**:manifest + theme-color + apple-touch,可「添加到主屏/安装」(完整离线 service worker 已于第五轮补齐,见 [public/sw.js](public/sw.js))。



- **修复卡片大小不一致**:[NodeCard.tsx](src/components/NodeCard.tsx) 加 `h-full` + 底部信息 `mt-auto`,同排卡片等高、底边对齐(5 列下尤其明显)。
- **地图彻底重做** [WorldMap.tsx](src/components/WorldMap.tsx):放弃平涂热力,改为 **`geo` 深色底图 + `effectScatter` 强调色脉冲光点**(质心定位、按节点数变大、全在线=强调色/有离线=琥珀);有节点的国家在底图上高亮;点击光点或国家弹节点列表。更像"活的"探针。
- **动态页面标题**:浏览器标签页标题跟随站名(呼应"标题很重要")。
- **加载骨架屏**:连接后端时显示脉冲占位卡片,替代单独转圈。
- **回到顶部悬浮按钮** [ScrollTop.tsx](src/components/ScrollTop.tsx):滚动一屏后出现。

## 第二轮

### 1. 实时汇率 API
- [src/utils/currency.ts](src/utils/currency.ts) + [src/hooks/useExchangeRates.ts](src/hooks/useExchangeRates.ts):接入免 key 的 `open.er-api.com`,`localStorage` 缓存 24h,失败回退静态汇率。
- 费用页底部显示「实时汇率 + 日期」或「静态估算」状态点。

### 2. Ping 图加高 + 时间范围
- [src/hooks/useNodeLatency.ts](src/hooks/useNodeLatency.ts) 参数化查询窗口 `windowMs`。
- [src/components/NodeDetail.tsx](src/components/NodeDetail.tsx) 延迟图高度 `h-56 → h-72/h-80`,新增 **30分 / 1时 / 6时 / 24时** 范围选择器(TCP 与 Ping 同步)。

### 3. 自定义右键菜单(替代禁用右键)
- 移除 App 的全局禁用右键;新增 [src/components/ContextMenu.tsx](src/components/ContextMenu.tsx)。
- **节点感知**(右键卡片/表格行):打开详情 / 加入对比 / 复制名称 / 复制 UUID。
- **全局动作**:刷新、卡片/表格/地图切换、费用统计、主题外观、回到顶部、全屏。
- 输入框内保留浏览器原生菜单;节点通过 `data-node-uuid` 标记识别。

### 4. laozig 品牌 + 每排 5 个 + 加宽
- 默认 `site_name` → `laozig 探针`,页脚 → `Crafted by laozig`;导航标题改为**渐变高亮字样**。
- 卡片模式每排最多 **5 个**(`xl:grid-cols-5`),主容器加宽到 `1600px`。

### 5. 侧边栏(超宽屏)
- 新增 [src/components/SideRail.tsx](src/components/SideRail.tsx):≥1750px 时左侧竖排品牌 `LAOZIG·STATUS` + 数据流装饰线,右侧实时迷你统计(节点/在线/地区/总流量)+ 实时时钟。窄屏隐藏。

### 6. 各页面打磨
- **费用页**:币种分布新增**月度成本占比堆叠条**(彩色),色点与图例联动。
- **表格模式**:状态从裸圆点 → **在线/离线胶囊**(带脉冲);新增 系统 / 运行时长 / 月费 等列(响应式隐藏);横向滚动 + 行 hover 主题色 + 表头大写字距。
- **地图模式**:热力色与发光标点**随强调色变化**;底色改为强调色调的主题化深色渐变;头部新增 地区/在线/离线 统计。
